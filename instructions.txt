This is instruction of pushing a local project (src) on Github

1.Create a Github repo to store the source code.
2.Copy the link of that repo you've created.
3.Git bash at this source code file.

3.1 git init  ( this command to set up ".git" file )

3.2 git status

3.3 git add .

3.4 git commit -m "your_message"

3.5 git remote add . https://github.com/toantomm/demo-local-push-Github.git

3.4 git push origin master
